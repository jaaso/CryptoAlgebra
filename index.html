CryptoAlgebra

Introductory Cryptography with a Bit of Abstract Algebra

Download this project as a .zip file Download this project as a tar.gz file

I am currently a math and computer science double major at the University of Arizona. As far as math goes, I have been getting into abstract algebra, and am taking my second course this semester. Additionally, I am in a cryptography class, which as it turns out, is a great application for ring theory.

I plan on talking about the crypto-algorithms I learn in class, and some of the theory behind them.

Code examples will be in Haskell, since the math-y foundations of Haskell play very nicely with algebra, and more importantly, why not.
